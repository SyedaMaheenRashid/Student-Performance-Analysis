# Student Performance Analysis

## Project Overview
This project analyzes student performance using Python. It explores data, checks missing values, creates new features, and visualizes results.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Steps Performed
- Checked missing values
- Analyzed outliers
- Created new features:
- Average Score
- Total Score
  
## Key Insights
- No missing values in dataset
- Reading and writing scores are strongly related
- Most students fall in average/good category

## Conclusion
This project demonstrates how data analysis and feature engineering help understand student performance.
