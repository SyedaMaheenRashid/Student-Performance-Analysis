#  Student Performance Analysis

##  Project Overview
This project focuses on analyzing student performance data to understand the key factors that influence academic success. Using data analysis and visualization techniques, we explore relationships between different variables such as gender, parental education, test preparation, and scores in different subjects.

The goal is to extract meaningful insights that can help improve student learning outcomes.

---

##  Objectives
- Analyze student performance data
- Identify factors affecting academic scores
- Draw meaningful conclusions from the dataset

---

##  Dataset Information
- Total Records: 1000 students  
- Features include:
  - Gender
  - Race/Ethnicity
  - Parental Level of Education
  - Lunch Type
  - Test Preparation Course
  - Math Score
  - Reading Score
  - Writing Score  

The dataset contains both **categorical and numerical variables**.

---

##  Data Cleaning
The following steps were performed:
- Checked for null values
- Verified data types
- Ensured dataset consistency
- Handled missing or inconsistent values (if any)

---

##  Exploratory Data Analysis (EDA)
The dataset was analyzed using Python libraries such as **Pandas, Matplotlib, and Seaborn**.

### Key analyses performed:
- Distribution of student scores
- Effect of test preparation course
- Correlation between subjects

---

##  Key Insights
- Students who completed test preparation scored higher on average
- Math scores show greater variability compared to other subjects
